# Import Best Practices

外部記事からベストプラクティスを抽出して反映するスキル。

## 実行例

```
❯ /import-best-practices この記事を取り込んで https://example.com/article

● import-best-practices を実行します。記事を取得し、公式と現状の設定を突き合わせて整合性を判定します。

  ## 抽出したプラクティス

  | # | 項目 | 公式整合 | 現状 | 評価 |
  |---|---|---|---|---|
  | 1 | X を Y するべき | ✅ | 未対応 | 取り込み推奨 |
  | 2 | A は B しないほうがよい | ✅ | 既に対応 | スキップ |
  | 3 | C を D に統一 | ❌ | 公式と矛盾 | 不採用 |

  どれを取り込みますか?
```
