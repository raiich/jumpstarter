# 動作環境別カスタマイズ

## Claude Code

- **並列走査**: カテゴリ数が多い・対象ファイルが多い場合、`Agent` ツール（`subagent_type=Explore`）でカテゴリ別に範囲指定して並列に走査する
- **構造化報告**: 検出結果はカテゴリ別に「検出 / 誤検知 / 未対処」を区別して 1 応答で提示する

## GitHub Copilot (CLI / VSCode)

- **並列走査**: 組み込みエージェント `Explore` を `agent` ツールで並列起動。対象が小さければ `grep` で逐次走査する
- **構造化報告**: 通常応答でカテゴリ別に列挙する
